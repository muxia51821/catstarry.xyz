## Known Decisions

### Blog
- Mixed content (tech + life + opinion), 2 weeks~1 month cadence
- Article skeleton: title → background (why) → process (what) → result (feeling/data)
- Writing flow: unstructured, no fixed template for "process" section
- Primary reader value: knowledge + inspiration
- Style reference: 安永全's raw personal energy + Airing's clean structure
- RSS, categories/tags, Giscus comments (no email notifications)
- Share buttons (X/Twitter, copy link, WeChat Moments)
- Page view count: backend only, user-visible only
- /blog and /feed are independent — no cross-linking

### Feed
- Unified timeline: both 碎碎念 + 剪藏 mixed, no public filtering
- Publishing: auth required (you only), floating "+" button (bottom right)
- Panel: default 碎碎念, tab switch to 剪藏; auto-link detection
- Post-publish: wait for uploads then appear; no editing, only hide (仅我可见, stays in timeline with marker) or delete
- Link preview: auto-fetch preferred, fallback to manual one-sentence summary
- Clip link card: title + link + commentary, expandable summary

### Finance
- Move from Excel to web-primary; Excel demoted to annual archive/signing record
- Core value over Excel: real-time market data API + PE thermometer + position deviation alerts
- Subdomain: f.catstarry.xyz (independent, not shown on homepage)
- Simple password auth for both 木下 (write) and cati (read-only)

### Design & Language
- Artistic, warm style (not minimalist). Use AI design tools (styles.refero.design, designmd.me, neuform.ai, getdesign.md) during design phase
- Phase 1: Chinese content/nav; Phase 2: add English UI toggle
