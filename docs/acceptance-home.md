# Home 星图入口验收清单

> 本清单只描述用户可观察的 Home 行为。星球材质、豹猫卫星和具体动效细节由 DESIGN.md 与 Phase 4 原型验证决定。

## 1. 宇宙入口

- [ ] 打开 catstarry.xyz，第一眼进入的是宇宙入口，不是 about 卡片、内容列表或混合时间线。
- [ ] 首屏能让访客理解可继续滚动，或可点击「DISCOVER MORE」继续探索。
- [ ] 首次滚动后，访客从远处接近同一片星域；2–3 个滚动阶段只表达空间纵深，不暗示栏目重要性。
- [ ] 远处的抽象星点在接近后逐步显现为具有体积、光照和不同地貌的完整星球，而不是永久的发光圆点或分类按钮。
- [ ] 首次滚动后出现低音量的侧边航行索引，可定位 Home 阶段，但不替代全站导航。

## 2. 自由星图总览

- [ ] 接近完成后出现 About、Blog、Feed、Learn、Projects 五颗自由分布的星球。
- [ ] 总览能看见五颗星球的完整全貌；聚焦或推进时能看到同一材质体系的高细节局部。
- [ ] 星球名称在总览时若隐若现；鼠标悬停或键盘聚焦时，名称和可进入状态更清楚。
- [ ] 星图默认保持静态，不自动循环旋转或持续抢夺注意力。
- [ ] 星球仅作为对应板块的入口与状态索引；Home 不展示文章、项目、笔记或时间线卡片。

## 3. 进入与导航

- [ ] 点击 Blog、Feed、Learn 或 Projects 星球，先有短暂推进感，再进入对应的功能页面。
- [ ] 每颗非 About 星球均可用键盘聚焦和触发。
- [ ] 首次滚动后的侧边航行索引能回到 Home 的入口、接近和星图阶段。
- [ ] 页面自然结束后进入页脚，不出现 Home 的 Recently 区块或内容瀑布流。

## 4. About 星球

- [ ] About 是星图中的一部分，不是中心天体、首屏卡片或独立信息流。
- [ ] 点击 About 星球或文字标签后，介绍直接在 Home 原地展开，不要求先发现豹猫，也不跳转至独立 About 页面。
- [ ] 豹猫卫星作为可选彩蛋：第一次点击蓄能，第二次点击后仅豹猫粒子局部爆开，并进入同一 About 展开态；About 星球本身不爆炸。
- [ ] About 展开态可关闭，并恢复到星图状态。
- [ ] 关闭由豹猫彩蛋打开的 About 后，粒子反向回收并恢复豹猫卫星；它不改变 About 作为普通星图成员的层级。

## 5. 非目标与数据边界

- [ ] Home 不请求或展示混合时间线、五源内容聚合、类型筛选、内容分页或无限滚动。
- [ ] Home 不展示 Feed 的公开足迹；公开足迹只在 /feed 浏览。
- [ ] 草稿、隐藏内容和来源模块的普通编辑不会以任何卡片形式进入 Home，也不会触发星球活动信号。

## 6. 星球活动信号

- [ ] Blog、Feed、Learn、Projects 星球只显示“活跃、稳定、沉寂”之一的低音量活动状态；Home 不显示任何活动内容。
- [ ] 最近 7 天内有符合规则的公开活动时，对应星球为活跃；超过 7 天且不超过 60 天时为稳定；超过 60 天或没有公开活动时为沉寂。
- [ ] Feed 星球同时参考公开碎碎念、剪藏和公开系统足迹；Blog、Learn、Projects 星球只参考各自符合规则的公开足迹。
- [ ] 活动状态不是访客未读提醒；不会显示事件数量、发布时间、标题、正文、摘要或列表。
- [ ] About 不参与活动状态；豹猫卫星不是活动信号。
- [ ] 活动卫星仅在进入视野、悬停或键盘聚焦时作一次克制反馈，不持续闪烁或自动循环。

## 7. 可访问性与设备适配

- [ ] 使用触控设备时，悬停所承载的关键信息可通过点按获得。
- [ ] 启用“减少动态效果”后，滚动与进入动效降级，不阻碍导航或阅读。
- [ ] 精细指针设备上，鼠标流星尾在 Home 清晰但克制；在 Content 页面更短更淡；Finance 与“减少动态效果”模式下关闭。
- [ ] 手机、平板与桌面端均能完成星图浏览、进入板块和 About 展开。

## 8. SEO

- [ ] 首页标题、描述和社交分享预览表达“catstarry 宇宙入口 / 星图”，而不是“数字生活混合时间线”。
- [ ] sitemap.xml 保留首页。
