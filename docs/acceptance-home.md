## 1. 首页整体

- [ ] 打开 catstarry.xyz，第一眼看到的是 about 卡片，然后是混合时间线
- [ ] 导航栏是科幻风格设计，和博客/碎碎念页面的导航栏明显不同

- [ ] about 卡片在页面最顶部
- [ ] 默认显示：3D 粒子风格画像 + 一句话自我介绍
- [ ] 点击后原地展开，显示完整自我介绍和动画

## 3. 混合时间线

- [ ] about 卡片下方是混合时间线，显示 15 条内容
- [ ] 内容包括五种类型：博客文章、碎碎念、剪藏、项目、学习笔记
- [ ] 按时间倒序排列（最新在最上面）
- [ ] 滚动到底部自动加载下 15 条（无限滚动）
- [ ] 草稿和隐藏内容不会出现在首页

## 4. 内容卡片区分

- [ ] 五种类型的内容用不同的卡片样式区分
- [ ] 每种卡片有彩色角标文字（博客/碎碎念/剪藏/项目/学习笔记）
- [ ] 不看角标也能从卡片样式大致判断类型
- [ ] 点击卡片跳转到对应的详情页面

## 5. 类型筛选（仅我自己）
