# 财务面板验收清单

> 木下验收用。按实际操作流程组织。不说技术术语。

## 1. 访问和登录

- [ ] 打开 f.catstarry.xyz，看到登录页面
- [ ] 用我的密码登录后，能录入交易、查看所有数据
- [ ] 用 cati 的密码登录后，只能查看数据，没有录入交易和修改的入口
- [ ] 登录后 12 小时自动过期
- [ ] 财务面板在 catstarry.xyz 首页完全不可见

## 2. 历史数据迁移

- [ ] 从 2026 年 5 月至今的所有 Excel 交易记录已导入面板
- [ ] Excel 中有公式错误的单元格被标记出来，我可以手动补填
- [ ] 导入后 Excel 只做年度归档，日常改用面板

## 3. 录入交易

- [ ] 点击面板上的「+」按钮，弹出录入窗口
- [ ] 填写：日期（默认今天）、标的名称、买入/卖出、数量、单价、仓位类别、原因（选填）
- [ ] 自动计算总金额（数量 × 单价），显示但不存储
- [ ] 提交后窗口不关闭，可以继续录入下一笔
- [ ] 手机也能流畅录入，输入框足够大

## 4. 查看持仓和行情

- [ ] 持仓面板显示所有持有标的：名称、数量、平均成本、当前价格、盈亏金额、盈亏百分比
- [ ] 价格每 15 分钟自动更新（有 15 分钟延迟可以接受）

## 5. PE 温度计

- [ ] 面板上看到 PE 温度计：五个色块（蓝/绿/黄/橙/红）
- [ ] 当前 PE 所在的档位高亮，并显示对应的操作建议
- [ ] PE 数据每天收盘后自动更新

## 6. 仓位偏离预警

- [ ] 每个仓位类别显示：目标占比、当前实际占比、偏离程度
- [ ] 超出上下限的仓位用红色标记
- [ ] 显示再平衡建议（如「卖出 X 份 沪深300，买入 Y 份 中证500」）

## 7. 三级熔断

- [ ] 回撤超过 10%，面板显示黄色横幅，暂停新增主动仓位
- [ ] 回撤超过 20%，面板显示红色横幅，强制只能定投宽基指数
- [ ] cati 提出重大异议时，我手动触发黑色熔断，全部暂停
- [ ] 黄色和红色触发时，cati 下次登录时收到通知
- [ ] 我可以手动解除熔断

## 8. 年度超额分成

- [ ] 12 月底，弹出确认窗口让我核实数据
- [ ] 系统自动计算超额分成（修正迪茨法 + 高水位线 + 年化 3% 基准）
- [ ] cati 可以看到完整计算过程
- [ ] 双方确认后，分成金额从我的年终奖投入中扣除

## 9. 年度复盘

- [ ] 我撰写年度总结
- [ ] cati 登录后看到总结，点击「确认签字」，时间戳记录
- [ ] 生成确认凭证（面板内可见）
- [ ] 一键导出全年数据为 Excel（含交易记录、快照、总结、签字记录）

## 10. cati 体验

- [ ] cati 用手机浏览器登录，看到和我一样的持仓、行情、PE 温度计、仓位偏离、操作建议

- [ ] 每月 1 号 cati 登录时，弹出确认窗口
- [ ] 我能查看 cati 的访问日志
