# 碎碎念与剪藏验收清单

> 木下验收用。按实际操作流程组织。不说技术术语。

## 0. 数据存储

- [ ] 碎碎念和剪藏的数据能正确存储和读取，不会丢失或乱码

## 1. 发布碎碎念

- [ ] 打开 /feed，右下角有一个「+」浮动按钮
- [ ] 点击「+」，弹出发布面板
- [ ] 输入文字（无字数限制），可选添加图片（最多 6 张，单张不超过 20MB）或视频（1 个，不超过 100MB 和 1 分钟）
- [ ] 图片和视频只能选一种，不能混选
- [ ] 选完图片或视频后自动开始上传，能看到上传进度
- [ ] 上传过程中如果某张图失败，不影响其他图，失败的有单独重试按钮
- [ ] 上传速度太慢时提示「网络较慢，是否继续等待？」
- [ ] 文件和文字都准备好后点「发布」，内容立刻出现在时间线上
- [ ] 如果发布时还有文件在上传中，发布按钮禁用，等上传完才能点
- [ ] 上传完成后能逐个删除文件

## 2. 发布剪藏

- [ ] 在发布面板切换到「剪藏」模式
- [ ] 粘贴一个网页链接，自动抓取标题、摘要和封面图
- [ ] 可以补充一段自己的点评
- [ ] 点「发布」，剪藏卡片出现在时间线上

## 3. 浏览时间线

- [ ] 打开 /feed，看到碎碎念和剪藏混合的时间线，按发布时间倒序
- [ ] 碎碎念卡片：文字内容在上面，图片以网格排列（1 张全宽、2 张并排、3 张并排、4 张宫格……），视频显示封面+播放按钮
- [ ] 剪藏卡片：标题（可点击跳转到原文）、摘要、封面缩略图、我的点评
- [ ] 点击图片放大查看，左右滑动翻页，点击外部关闭
- [ ] 点击视频封面播放/暂停

## 4. 登录和权限

- [ ] 未登录时，「+」按钮隐藏，只能浏览时间线
- [ ] 点击右下角「登录」，输入用户名和密码后登录成功
- [ ] 登录后 12 小时自动过期，需重新登录

## 5. 管理后台

- [ ] 登录后能看到「管理」入口，点击进入独立管理页面
- [ ] 管理页面显示所有帖子列表（公开的 + 仅我可见的），按时间倒序
- [ ] 可以按状态（公开/仅我可见）、类型（碎碎念/剪藏）、日期范围筛选
- [ ] 可以勾选多条帖子，批量隐藏或批量删除
- [ ] 隐藏后的帖子只有我在管理后台能看到，时间线上不显示
- [ ] 已隐藏的帖子可以恢复公开
- [ ] 删除需要二次确认，删了就没了

## 6. 手机浏览和发布

- [ ] 用手机打开 /feed，时间线排版正常
- [ ] 发布面板从底部弹出，占屏幕三分之二，触控区域足够大
- [ ] 手机也能完成碎碎念发布和剪藏发布全流程
