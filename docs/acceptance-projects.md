# 项目展示验收清单

> 木下验收用。按实际操作流程组织。不说过多技术术语。

## 1. 浏览项目页面

- [ ] 打开 /projects，看到项目卡片网格布局
- [ ] 每张卡片显示：项目截图、项目名称、一句话描述、技术标签
- [ ] 点击卡片跳转到项目的外部链接

## 2. 项目在首页时间线

- [ ] 打开首页，能看到项目卡片（显示最近 1 个项目）
- [ ] 项目卡片显示：截图、项目名称、描述、彩色角标「项目」
- [ ] 点击跳转到 /projects

## 3. 新项目上架

- [ ] 我告诉 AI agent 有新项目要上架
- [ ] AI agent 按模板添加项目索引信息
- [ ] Git push 后自动部署，新卡片出现在 /projects
