# HAS-02 — Map Eligible Public Activity to Each Planet

**Parent:** Home Activity Signal PRD

**What to build:** 定义 Blog、Feed、Learn、Projects 与 About 的活动来源关系，使 Home 的星球状态只反映符合既有 Public Footprint 与公开时间线规则的事件。

**Blocked by:** HAS-01 — Define Home Activity State Semantics.

## Acceptance criteria

- [ ] Blog 星球只根据最新公开 Blog 发布足迹判断状态。
- [ ] Learn 星球只根据最新公开的“完成一个小节”足迹判断状态。
- [ ] Projects 星球只根据最新公开、显式标记为实质更新的足迹判断状态。
- [ ] Feed 星球根据 Unified Public Timeline 的最新公开事件判断状态，包含公开碎碎念、剪藏与公开系统足迹。
- [ ] 某次 Blog、Learn 或 Projects 的系统足迹可同时使其来源星球与 Feed 星球反映活动；这不是 Home 内容聚合。
- [ ] About 不参与活动状态，也不会被显示为沉寂。
- [ ] 隐藏或私密事件不作为任何星球的公开活动来源。

## Agent brief

本卡只定义“什么事件有资格影响哪个星球”。不要改变 Feed 的时间线职责、不要创建历史事件、不要推测 Projects 实质更新。
