# HAS-03 — Specify the Home Static Activity Projection Contract

**Parent:** Home Activity Signal PRD

**What to build:** 定义 Home 消费的最小活动状态投影，使 SSG 星图只获得四颗功能星球的状态，而不重新取得具体公开内容。

**Blocked by:** HAS-01 — Define Home Activity State Semantics; HAS-02 — Map Eligible Public Activity to Each Planet.

## Acceptance criteria

- [ ] Home 可取得 Blog、Feed、Learn、Projects 各自的活动状态，且 About 不在状态投影中。
- [ ] 投影不包含标题、正文、摘要、链接、列表、时间线、事件数量或精确活动时间。
- [ ] Home 不需要请求 `/api/home`，也不直接读取 Public Timeline 或来源模块内容。
- [ ] Home 继续是 SSG 宇宙入口与星图导航；状态只驱动低音量信号卫星，不改变各星球的导航职责。
- [ ] 投影失败或过期时的用户可见降级行为被明确交给定向 Phase 3 裁决。
- [ ] 具体生成时机、数据读取位置、刷新机制与是否新增 ADR 由定向 Phase 3 裁决，不在本卡预设实现。

## Agent brief

优先把它理解为受控静态投影契约，而不是新的 Home 内容 API。保持 ADR-006 的退役范围：不得恢复混合时间线、`HomeTimelineItem`、KV bridge 或五源内容聚合。
