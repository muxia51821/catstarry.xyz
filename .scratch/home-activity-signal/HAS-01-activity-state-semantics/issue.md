# HAS-01 — Define Home Activity State Semantics

**Parent:** Home Activity Signal PRD

**What to build:** 为四颗功能星球定义一致、可理解的 `active`、`stable`、`dormant` 活动状态，使 Home 能表达公开活动的时间感，而不变成消息中心或内容流。

**Blocked by:** None — can start immediately.

## Acceptance criteria

- [ ] 最近 7 天内有合资格公开活动的星球被定义为“活跃”。
- [ ] 超过 7 天且不超过 60 天有合资格公开活动的星球被定义为“稳定”。
- [ ] 超过 60 天，或从未有合资格公开活动的星球被定义为“沉寂”。
- [ ] 16 到 60 天继续属于“稳定”，不会新增第四种用户可见状态。
- [ ] 三态表达板块公开活动，不表达访客是否读过内容。
- [ ] 私密、隐藏、删除、草稿、普通编辑、失败部署和重复部署不会制造活动状态。
